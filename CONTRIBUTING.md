# Contributing to Indhra's Portfolio

Thank you for your interest in this portfolio website!

## Can I Contribute?

### What You CAN Do:
- ✅ **View the code** — This is a public repository
- ✅ **Fork the repository** — Create your own copy
- ✅ **Raise Pull Requests** — Submit your proposed changes

### What Only the Owner Can Do:
- ❌ **Accept/Merge Pull Requests** — Only @indhra (the repository owner) can merge PRs
- ❌ **Push directly to main branch** — Write access is restricted to the owner

## How to Submit a Pull Request

If you find a bug, typo, or have a suggestion:

1. **Fork** this repository
2. **Create a branch** in your fork:
   ```bash
   git checkout -b fix/your-fix-name
   ```
3. **Make your changes** and commit:
   ```bash
   git commit -m "Brief description of your change"
   ```
4. **Push to your fork**:
   ```bash
   git push origin fix/your-fix-name
   ```
5. **Open a Pull Request** to the `main` branch of this repository

The repository owner will review your PR and decide whether to merge it.

## Guidelines

- Keep changes focused and minimal
- Test your changes locally before submitting
- Provide a clear description of what your PR does
- Be respectful and patient while waiting for review

## Questions?

If you have questions about the repository permissions or PR process, feel free to open an issue.
